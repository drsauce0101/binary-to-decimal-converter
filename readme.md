# Bin2Dec - Conversor de Binário para Decimal

Um conversor simples e elegante de números binários para seus equivalentes decimais, construído com HTML, CSS e JavaScript puro. Este projeto faz parte dos desafios do repositório "App Ideas".

## ✨ Funcionalidades

- Converte números binários de até 8 dígitos.
- Validação de entrada para aceitar apenas os caracteres '0' e '1'.
- Design limpo e responsivo com um "card" centralizado.
- Assinatura personalizada no rodapé.
